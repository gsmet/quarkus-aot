# Quarkus - Experimenting with Project Leyden

This repository contains some sample Quarkus applications to experiment with Project Leyden.

Each subdirectory contains a specific application, together with scripts.
Have a look at each `README.md` for more details.

| Application | Description |
|----------|----------|
| quarkus-simple-rest-aot  | A simple minimal REST app using Quarkus REST  |
| quarkus-hibernate-orm-simple  | A simple Hibernate ORM based app with one entity  |
| quarkus-hibernate-orm-spacefox  | A full Hibernate ORM CRUD app with 1k entities, repositories, mappers, services, REST resources...  |
| quarkus-hibernate-orm-tribe-krd  | A Hibernate application based on https://github.com/topicusonderwijs/tribe-krd-quarkus |
